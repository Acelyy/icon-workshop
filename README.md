## 图标工场

移动应用图标生成工具，一键生成所有尺寸的应用图标。

https://icon.wuruihong.com

### 功能

* 生成不同移动平台（iOS/Android/Web App/Cordova/...）的应用图标，一键生成，快捷导入
* App 启动图（Launch Image/Splash）生成工具，支持 iOS/Android

### 技术栈

* Laravel 5.6
* Angular 1 + Material Design
* Vue / Quasar Framework
